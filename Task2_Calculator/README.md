# Task 2 - Simple Calculator 

# Description:
A basic calculator built using C++ that can perform the following operations:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

##  Features:
- User input for two numbers
- Operator input (+, -, *, /)
- `switch-case` based logic
- Handles division by zero error
